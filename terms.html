2. Service Description

Remocat is a mobile application that provides advanced facial recognition authentication features. With Remocat, you can secure your devices and sensitive content with a facial recognition unlocking system designed to offer the highest level of security.

The app functions as a secure authentication method for mobile devices, protecting access to sensitive features via facial biometric recognition. No password is required, meaning that only you, and your face, will have access to what you wish to protect.

Remocat also integrates an advanced facial recognition module based on TensorFlow Lite, a machine learning technology that runs locally on the device. This system allows the app to compare the face of the person unlocking the device with a registered facial model of the authorized user in real time. This makes it possible to automatically detect unauthorized access. The process is completely offline: no biometric data is transmitted or stored on external servers.
